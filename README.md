# Habeeba Faiza — Personal Branding Website

Modern personal branding website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- GitHub Pages (via GitHub Actions)

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
Push to `main`; workflow [.github/workflows/deploy.yml](.github/workflows/deploy.yml) deploys to GitHub Pages.
